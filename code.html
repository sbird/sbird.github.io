Code

The code described below can be found on my github account, as well as a number of other useful scripts.

I made a version of the bibtex style file for MNRAS, mn2e.bst, which does not crash on papers with more than 8 authors, and also allows the use of hyperlinks to arxiv and ADS papers: mn2e.bst. The fixed style file is by John Sleath and Tim Jenness. The original hyperlinks were added by Antony Lewis.

DLA analysis scripts [Download]
A collection of scripts for analysing observables related to DLAs in Gadget/Arepo simulations. Uncludes routines for, eg, calculating a column density function. The basic unit of analysis is a projected density grid.
Fourier-space neutrino patches [Download]
Patches to Gadget-3 which implement the Fourier-space neutrino method of 1209.0461. The code can be enabled by setting the KSPACE_NEUTRINOS_2 flag at compile-time. Will not work with the public version of Gadget-2 (yet), as it uses the built-in power spectrum estimator.
CAMB Neutrino Patches [Download]
Three patches for CAMB that allow it to reasonably accurately measure the impact of massive neutrinos on the non-linear matter power spectrum. Incorporated in the official CAMB, as of December 2011.
Matter Power Spectrum Estimator [Download] [Download from github]
Extracts the baryon and dark matter power spectra from a Gadget-II snapshot file. Written in C++, parallelized with OpenMP. Requires an FFTW3 compatible FFT library and GadgetReader
Gadget Reader Library [Download] [Download from github]
A tiny C++ library for reading Gadget Format files. Has a somewhat higher-level interface than the readgadget.c that comes with Gadget itself. In particular, it is able to transparently handle multiple files in one snapshot, check the consistency of headers, swap endianness and will attempt to transparently read files in Gadget-I format, while attempting to minimise file reads. A version of this library re-written in python is incorporated into pynbody
DBI Simulator [Download]
The DBI simulator for this paper, which calculates the trajectory of the inflaton in DBI inflation, given some initial conditions. For this to compile, you will need to supply an ordinary DE solver. The one in Numerical Recipes works very well.
